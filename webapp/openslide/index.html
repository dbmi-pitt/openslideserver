OpenSlide Server (v3.2.5)

OpenSlide Server is a Whole Slide Image (WSI) server that uses OpenSlide library to read WSI files. OpenSlide is a C library that provides a simple interface to read whole-slide images (also known as virtual slides).

OpenSlide Server has a RESTful API to access virtual slides hosted on that server. One typically needs to have an OpenSlide Viewer client to open WSI files hosted by OpenSlide Server. The direct URL to OpenSlide Server is http://ties-vm/openslide/server

OpenSlide Server was developed by Eugene Tseytlin in the Crowley-Jacobson Lab at the Department of Biomedical Informatics, University of Pittsburgh.




requires Java 1.5 or higher to run
